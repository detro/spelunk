# Base64 Encoder Secret Modifier Alias (`b64e`)

This modifier is simply an alias for the [`b64`](../base64/README.md) Base64 Encoder Secret Modifier. 

It exists to provide a symmetrical naming convention (`b64e` for encode, `b64d` for decode) while preserving backward compatibility with the original `b64` modifier.

Please refer to the [Base64 Encoder Secret Modifier Documentation](../base64/README.md) for full usage and behavior details.
